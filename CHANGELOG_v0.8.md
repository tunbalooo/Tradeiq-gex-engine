# TradeIQ v0.8 — Interactive Chart Workstation

- Preserves the classic dashboard and all backend engines.
- Replaces only the dashboard chart renderer and the full Chart page.
- Uses TradingView Lightweight Charts 5.2.0 with required attribution.
- Adds mouse-wheel zoom, drag-to-pan, price-scale zoom, recenter, fit, real-time, auto-scale and fullscreen controls.
- Adds candlestick/line switching, normal/magnet crosshair modes and user horizontal lines.
- Keeps TradeIQ overlays: EMA 9/21/55, GEX walls/flip, Fib/OTE, supply/demand, VWAP/standard deviation and entry/SL/TP.
- No confidence weights, setup logic, Databento logic, session gate or database code changed.
