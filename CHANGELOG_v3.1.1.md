# TradeIQ v3.1.1 — Flexible Institutional Cluster Tiers

- Exceptional two-factor clusters can now qualify with strict confirmation, confidence and execution-freshness requirements.
- Three-factor clusters use the standard institutional gate.
- Four-or-more-factor clusters receive high-priority composite selection without bypassing risk or execution safety.
- Related evidence remains grouped to prevent false confidence.
- The Decision Brain evaluates both the cluster and strongest single model, then uses the strongest actionable interpretation.
- UI and API now expose cluster tier, selection score and active independent categories.
